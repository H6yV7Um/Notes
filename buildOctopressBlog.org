本文是在另一台PC（该PC上已安装了ruby 1.9.3）上，为已存在的octopress博客 username.github.com 搭建博客环境。

- 安装 msysgit, 一路回车
- 打开新安装的 git Bash
  - 在github相应网站的邮箱设置中，增加邮箱 companyEmailAddr 

    一定要为github验证该邮箱，否则，在新增一个博文时，尽管rake preview能更新网址 localhost:4000，但是，rake gen\_deploy不能更新网址 username.github.com
 - 为邮箱 companyEmailAddr 产生密钥
   - =ssh-keygen -C companyEmailAddr=
   - 将产生的公钥复制添加到github网站的密钥设置中
 - =git clone git@github.com:tech-note.github.com.git=
 - =git config --global user.name username4githubsite=
 - =git config --global user.email companyEmailAddr=
 - =gem install bundler=
 - =cd ./username.github.com=
 - =bundler install=
 - =rake generate=
