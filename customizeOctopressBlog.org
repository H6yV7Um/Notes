* 评论功能
** 申请disqus账户
- 填写邮箱
- shortname
- website URL: http://githubUsername.github.com
** 修改\_config.yml
在字段disqus\_short\_name:后填写上面disqus申请时的shortname，注意空格是必须的。


* 添加侧边栏 categories 和 tag cloud
** octopress-tag-pages.git
#+begin_src sh
$cd /home
$git clone git@github.com:robbyedwards/octopress-tag-pages.git
#+end_src
- 复制 tag\_generator.rb 到 ./octopress/plugins 目录
- 复制 tag\_index.html 到 ./octopress/source/\_layouts 目录
- 复制 tag\_feed.xml 到 ./octopress/source/\_includes/custom/ 目录
** octopress-tag-cloud.git
但是该 git@github.com:robbyedwards/octopress-tag-cloud.git 对UTF-8字符的支持有问题，[[http://pfmiles.github.com/blog/liquid-error-about-regexp-match-when-using-octopress-tagcloud/][详见]]， 所以，直接使用了已解决了此问题的 git@github.com:alswl/octopress-category-list.git。

=$git clone git@github.com:alswl/octopress-category-list.git=

** 创建 tags.html 和 category\_list.html
创建位置： ./octopress/source/\_includes/custom/asides/。

tags.html 内容:
#+begin_src html
<section>
  <h1>Tags</h1>
  <ul class="tag-cloud">
    {% tag_cloud font-size: 90-210%, limit: 10, style: para %}
  </ul>
</section>
#+end_src

Category_list.html 内容:
#+begin_src html
<section>
  <h1>Categories</h1>
  <ul id="categories">
    {% category_list %}
  </ul>
</section>
#+end_src

** 修改\_config.yml
#+begin_src sh
default_asides: [asides/category_list.html, asides/recent_posts.html, custom/asides/tags.html]
#+end_src
